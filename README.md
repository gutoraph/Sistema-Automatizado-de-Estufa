# Uso e Funcionamento
O projeto tem como funcionalidade em automatizar as estufas e deixa-las regadasm, minimizando o esforço humano.

Montamos os componentes de acordo com o circuito eletrônico que está localizado nesse [repositório](https://github.com/gutoraph/Sistema-Automatizado-de-Estufa/blob/525c94e620d57a445c59d9ee975ddc0be01ae192/circuito%20eletronico.png).



# Componentes Utilizados
• Arduino Uno R3 <br />
• Sensor de umidade do solo higrômetro <br />
• Sensor de temperatura DS18B20 <br />
• Sensor de luminosidade LDR <br />
• Protoboard <br />
• Mortor DC 3-6V <br />
• Bateria 9V <br />
• 2 Resistor <br />
• 11 Jumpers Macho-Macho <br />
• 6 Jumpers Macho-Fêmea <br />
• Ponte-H L298N
